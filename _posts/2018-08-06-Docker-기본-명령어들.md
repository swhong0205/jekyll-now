## docker build
도커 이미지를 만들기 위해서는 build 명령어를 사용합니다.

```
$ docker build -t imagename:tagname .
```

## docker images
생성된 이미지를 확인하려면 images 명령어로 확인할 수 있습니다.

```
$ docker images
```

## docker run
생성된 이미지로 컨테이너를 만드려면 run 명령어를 사용합니다.
이미지를 실행한다고 생각하면 편함.

```
$ docker run -p 8080:8080 imagename
```
